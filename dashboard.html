Export Guides
IFC File (Required)

The IFC file contains all piece marks, locations, member properties, elevations, and grid coordinates. This is the most important file.

SDS2 / Design Data
Home → Export → 3D → Export Model → IFC4 → Save
Make sure IFC4 is selected (not IFC2x3).
Tekla Structures
File → Export → IFC → Select IFC4 → Export
Use the "Steel" export filter for best results.
RISA-3D / Other Software
File → Export → BIM/IFC → IFC4 format → Save
Contact support@steelcaller.com if your software is not listed.
KSS File (Recommended)

The KSS file provides full bolt specifications, attached material, plate sizes, and assembly weights. Highly recommended for complete field data.

SDS2 / Design Data
Home → Export → MRP Interface → KiSS Software → Export
Accept defaults. The .kss file will be saved to your export folder.
NC1 / DSTV Files (Recommended)

NC1 files provide per-end hole counts and section dimensions for each piece. Zip all .nc1 files into a single .zip before uploading.

SDS2 / Design Data
Home → Export → CNC Download → DSTV → Select all members → Export
Select .nc1 format. Then zip all exported .nc1 files into one folder and upload the .zip.
💡 Pro Tip: Export all three files right after your model is complete and approved. Re-export and upload a new revision any time the model changes — the field crew sees revision alerts automatically.
