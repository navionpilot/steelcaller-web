Upload Your Model Files

Export these files directly from your steel modeling software. The more files you upload, the richer the field app experience for your crew.

IFC File
.ifc
Required
Piece marks, locations, connections, elevations, member types, and weights
KSS File
.kss
Recommended
Full bolt specifications, attached material detail, plate sizes, and assembly weights
NC1 Files
.zip
Recommended
Per-end bolt counts and section dimensions. Zip all .nc1 files into one folder before uploading
Export IFC from your modeling software via Home → Export → 3D → Export Model → IFC4. Export KSS via Home → Export → MRP Interface → KiSS. Zip all NC1 files from Home → Export → CNC Download → DSTV → .nc1.
