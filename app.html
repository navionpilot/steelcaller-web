What Is SteelCaller
Your Complete Field Lookup Tool
SteelCaller gives ironworkers, foremen, and superintendents instant access to every piece mark on the job — right from their phone. No paper drawings, no radio calls to the office, no waiting.

Search any piece mark and instantly see its section size, crane pick weight, location, bolt specs, connection ends, revision status, and camber direction — everything needed to set iron correctly the first time. Say "Call 52B1" and the AI voice reads it all to you hands-free.
💡
No app download required. Open the link your fabricator sent you in any browser on your phone. Tap the Share button and "Add to Home Screen" to install it like an app for quick access every morning.
2
How to Access Your Project
You Were Sent a Link
Your fabricator uploaded the project and sent you a unique project link. It looks like this:
steelcaller.com/projects/5665-parkway-support
Open that link on your phone. The app loads immediately — no login, no password, no account needed. Anyone with the link can access the project.

Search Methods
Search by Piece Mark
Type the piece mark exactly as it appears on the steel — 52B1, 22C1, 37B1. Results appear instantly as you type. Tap any result to open the full piece card.
Type: 52B → shows all marks starting with 52B
Search by Grid Line
Type a grid line identifier and the app shows every piece at that grid. Great for planning a full column line or bay.
Type: C3 → shows all pieces at grid C3
Type: C → shows all pieces on grid line C
Search by Elevation
Type an elevation and see everything at that level. Useful when you're working a specific floor and want to see all marks in your area.
Type: 14 → shows pieces at El 14'-0"
Type: EL 28 → shows pieces at El 28'-0"
⚡
AI Smart Search understands partial marks, grid ranges, and elevation ranges. You don't need to type exactly — just type what you know and it finds the closest match.
Filter Pills
Use the filter pills below the search bar to narrow results instantly:

Seq 3, Seq 4 — show only pieces in that sequence
Grid C — show only pieces on grid line C
El 14'-0" — show only pieces at that elevation
Revised ⚠ — show only marks that have been revised
Missing ⚠ — show only marks in the active sequence not yet marked Complete
Reading the Piece Card
Revision Alert — Red Banner at Top
If the fabricator uploaded a revised model, any changed piece shows a red REVISED banner at the very top of the card. It tells you exactly what changed — section size, length, bolt count, etc. Always check this before you pick the piece.
Camber Direction — Orange Banner
Any beam with camber shows a large orange CAMBER UP banner with the camber amount. This beam must be installed with the crown facing up. Installing it upside down means the beam comes back down. The camber value is also shown in the member properties grid.
Crane Pick Weight
Shows the full assembly weight — main member plus all shop-welded plates, angles, clips, and studs. This is the actual weight the crane is picking, not just the bare section weight. Always use this number for your crane pick, not the section weight alone.
Connection Ends — Left & Right
Shows which mark each end connects to and the exact field bolt specification for that connection — bolt diameter, length, grade, count, and washer requirement. Bolt data comes from both the KSS material list and the NC1 CNC files for accuracy.

Beams: Left End / Right End
Columns: Bottom / Top
Braces: Lower End / Upper End
Report Field Issue Button
Tap this to log a field problem on this mark — wrong piece, doesn't fit, damaged, missing material. The report is timestamped and sent directly to the fabricator and superintendent. Include a photo for fastest resolution.

Revision Alerts
What Triggers a Revision Alert
When the fabricator uploads a new version of the model, SteelCaller automatically compares the new data to the previous version. Any piece where something changed gets flagged with a red REVISED banner.

Changes that trigger an alert:
• Section size changed (W16x40 → W18x40)
• Length changed
• Bolt count or spec changed
• Connection end changed
• Grade changed
⚠️
Always check the Alerts tab first thing in the morning. If a piece was revised overnight and you set it before checking, it has to come back down. That's expensive.
Missing Piece Alerts
When a piece mark in the current erection sequence has not been marked Complete, the app tracks it as outstanding. If a piece from an active sequence has not been marked Complete, it shows as a Not Complete alert on the dashboard. This prevents the crane from sitting idle waiting for a piece nobody knew was missing.
AI Pick Order Recommendation
How It Works
The AI looks at the current erection sequence, what pieces have been marked Complete, and the crane's last known position — then recommends the most efficient pick order to minimize crane repositioning and maximize pieces set per hour.

The recommendation is a suggestion, not a requirement. The foreman always has final say. Tap "Use This Plan" to share it with the crew or "Dismiss" to ignore it.
Required & Recommended Files

.ifc — IFC4 File
REQUIRED
.kss — KiSS File
RECOMMENDED
.nc1 Files — ZIP
RECOMMENDED
How to Export Each File from Your Modeling Software
IFC4 File — Required
Contains all piece marks, locations, connections, elevations, grids, and 3D geometry.
1
Open your project in your steel modeling software
2
Go to Home → Export → 3D → Export Model
3
Set Export file format to IFC4
4
Set output destination and click OK
5
Upload the resulting .ifc file to SteelCaller
KiSS File (.kss) — Recommended
Adds complete field bolt specifications, plate sizes, and full assembly weight calculations for accurate crane picks.
1
Go to Home → Export → MRP Interface
2
Select KiSS as the export format
3
Set output destination and click OK
4
Upload the resulting .kss file to SteelCaller
NC1 Files (.nc1 ZIP) — Recommended
Adds per-end bolt counts, section dimensions, and cope geometry. One file per piece mark — zip the folder before uploading.
1
Go to Home → Export → CNC Download
2
Set CNC type to DSTV and file extension to .nc1
3
Select all members and click OK
4
Select all .nc1 files in the output folder and zip them into a single .zip file
5
Upload the .zip file to SteelCaller
💡
All three files are standard exports that most steel modeling programs already generate for fabrication. No extra modeling work is required.
